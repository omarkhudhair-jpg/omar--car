# طريقة بسيطة لرفع التطبيق

## الحل الأسهل: استخدام GitHub Desktop

بدلاً من سطر الأوامر المعقد، يمكنك استخدام GitHub Desktop (برنامج بواجهة رسومية):

### الخطوة 1: تحميل GitHub Desktop
1. اذهب إلى: https://desktop.github.com/
2. حمّل البرنامج وثبته
3. سجل دخول بحساب GitHub الخاص بك

### الخطوة 2: إضافة المستودع
1. في GitHub Desktop، اضغط File → Add Local Repository
2. اختر مجلد `OMAR CAR`
3. اضغط Add Repository

### الخطوة 3: النشر
1. اضغط Publish Repository
2. تأكد أن الاسم `omar-car`
3. تأكد أن Public محدد
4. اضغط Publish

### الخطوة 4: تفعيل GitHub Pages
1. اذهب إلى صفحة المستودع على GitHub
2. Settings → Pages
3. اختر Branch: main
4. احصل على الرابط

---

## أو: دعني أرفعها لك الآن مباشرة

إذا كنت تريد، يمكنني رفع الملفات الآن باستخدام Git من الكمبيوتر.

فقط أخبرني: **ما هو اسم المستخدم الذي اخترته في GitHub؟**

مثال: إذا كان اسمك `ahmed123`، أخبرني به وسأرفع كل شيء تلقائياً.
